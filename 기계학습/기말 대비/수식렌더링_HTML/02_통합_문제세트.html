1. AdaBoost 의사코드 핵심

  1. Training set \(S=\{(x_i,y_i)\}\)
  2. 초기 sample weight/distribution D1을 uniform으로 설정
  3. \(t = 1 ... T\) 반복
  4. 현재 분포 Dt로 weak learner \(h_t\) 학습
  5. weighted error \(\epsilon_t\) 계산
  6. \(\epsilon_t\ge0.5\)이면 weak learner로 부적절하므로 버리거나 중단
  7. learner 신뢰도 \(\alpha_t\) 계산
  8. 오분류 샘플 weight 증가, 정분류 샘플 weight 감소
  9. weight normalization
  10. 최종 예측은 \(\mathrm{sign}\left(\sum_t \alpha_t h_t(x)\right)\)

시험 포인트: - \(D_t\): 샘플이 뽑히거나 중요하게 반영될 분포 - \(\epsilon_t\): 현재 weak learner의 weighted error - \(\alpha_t\): learner의 상대적 신뢰도 - sign: binary label을 최종 결정 - 반복 횟수 T와 실제 채택된 learner 수는 다를 수 있음

4. SHAP 답안 틀

SHAP은 특정 관측치의 예측값을 설명하기 위해 각 feature를 협력 게임의 player로 보고 Shapley value를 계산한다. 각 feature의 contribution을 더하면 예측값 설명이 되고, LIME처럼 surrogate additive model 관점을 가지지만 cooperative game theory 기반이라 local accuracy, missingness, consistency 같은 성질을 만족한다.

5. 2026-05-20 XAI 답안 틀

\(f(x)\)와 g(x)

\(f(x)\)는 학습된 예측 모델이고, g(x)는 그 예측을 사람이 이해할 수 있도록 근사하거나 설명하는 설명 모델이다. Post-hoc XAI에서는 \(f(x)\)를 먼저 만든 뒤 g(x) 또는 별도 설명 방법으로 \(f(x)\)의 예측 근거를 해석한다.

PDP

PDP는 관심 feature를 변화시키고 나머지 feature를 marginalization하여 평균 예측이 어떻게 변하는지 보여준다. 따라서 global한 평균 효과를 설명하는 데 좋지만, sample별 이질성이나 feature interaction을 가릴 수 있다.

ICE

ICE는 각 instance마다 관심 feature 변화에 따른 prediction curve를 그린다. PDP가 평균 curve라면 ICE는 individual curve 모음이다. Heterogeneous relation을 볼 수 있지만 sample이 많으면 그래프가 overcrowded될 수 있다.

Morris

Morris sensitivity analysis는 다른 feature를 고정하고 하나의 feature만 변화시키는 one-at-a-time 방식이다. \(\mu\)는 평균적 영향력, \(\sigma\)는 영향이 얼마나 흔들리는지 또는 interaction/nonlinearity 가능성을 나타낸다.

Permutation Importance

Permutation importance는 특정 feature 값을 섞어 해당 feature의 정보를 깨뜨린 뒤 성능 하락을 측정한다. 성능이 크게 떨어질수록 그 feature가 중요하다고 해석한다.

B. 주관식

문제 9

Boosting과 Bagging의 차이를 설명하시오.

모범답안: Bagging은 bootstrap sample로 여러 weak learner를 대체로 독립적으로 학습해 variance를 줄이는 앙상블이다. Boosting은 이전 learner가 틀린 샘플 또는 residual에 다음 learner가 더 집중하도록 순차적으로 학습해 strong learner를 만든다. 따라서 Boosting은 learner 간 의존성이 있고 병렬화가 어렵다.

문제 10

AdaBoost 의사코드의 핵심 흐름을 설명하시오.

모범답안: 초기에는 모든 샘플에 균등한 weight를 준다. 반복마다 현재 weight 분포로 weak learner를 학습하고 weighted error를 계산한다. error가 낮은 learner는 큰 alpha를 받고, 오분류 샘플의 weight는 증가한다. 마지막에는 각 learner의 예측을 alpha로 가중합한 뒤 sign으로 최종 class를 결정한다.

문제 11

Gradient Boosting을 Functional Gradient Descent 관점에서 설명하시오.

모범답안: 일반적인 gradient descent가 숫자 weight를 업데이트한다면, gradient boosting은 기존 모델이 설명하지 못한 residual 또는 loss를 줄이는 방향으로 새로운 weak learner라는 function을 추가한다. 이 과정을 반복해 ensemble function이 점진적으로 loss를 줄이도록 만든다.

문제 12

SHAP이 LIME과 비슷하면서도 다른 점을 설명하시오.

모범답안: 둘 다 특정 예측 주변을 설명하기 위해 단순한 additive surrogate explanation 관점을 가진다. 하지만 SHAP은 Shapley value와 cooperative game theory에 기반해 각 feature의 공정한 기여도를 계산하며, local accuracy, missingness, consistency 같은 성질을 만족한다. LIME은 더 직관적이고 범용적이지만 이런 성질을 모두 보장하지 못한다.

문제 13

SHAP summary plot에서 확인할 수 있는 정보를 쓰시오.

모범답안: 각 feature의 전반적 중요도, feature 값이 high/low인지, 그 값이 예측을 증가 또는 감소시키는 방향, 그리고 관측치별 SHAP value의 분포를 함께 볼 수 있다. 따라서 단순 feature importance보다 local contribution의 방향성과 산포를 더 잘 보여준다.

문제 14

XGBoost가 기존 Gradient Boosting Machine에서 보완한 점을 설명하시오.

모범답안: XGBoost는 GBM의 순차적 residual minimization 구조를 유지하되 목적식에 regularization term을 추가해 과적합을 줄인다. 특히 leaf 수에 대한 gamma penalty와 leaf weight의 L2 regularization을 사용하고, split gain이 충분하지 않으면 pruning한다. 또한 병렬화, 캐시, 메모이제이션 같은 계산 효율 개선도 포함한다.

문제 15

CatBoost가 categorical feature를 다룰 때 target leakage를 줄이는 방식을 설명하시오.

문제 17

PDP와 ICE를 비교해 설명하시오.

모범답안: PDP는 관심 feature를 변화시킬 때 모델 예측이 평균적으로 어떻게 변하는지 보여주는 global한 해석 도구다. 나머지 feature는 marginalization되어 평균 효과가 드러난다. 반면 ICE는 각 관측치별로 feature 변화에 따른 예측 변화를 따로 그려 heterogeneous relation을 더 잘 보여준다. 다만 관측치가 많으면 그래프가 복잡해진다.

문제 18

LIME의 기본 절차를 설명하시오.

모범답안: \(f(x)\)는 이미 학습된 예측 모델이고, g(x)는 그 예측을 설명하기 위한 설명 모델 또는 설명 방법이다. Post-hoc XAI에서는 먼저 black-box model f를 학습해 예측 성능을 확보한 뒤, f가 어떤 feature에 근거해 예측했는지를 g나 PDP, ICE, LIME, SHAP 같은 별도 방법으로 해석한다. 따라서 예측 모델 자체를 바꾸는 것이 아니라 예측 결과를 사람이 이해할 수 있게 만드는 과정이다.

문제 54

PDP의 장점과 한계를 설명하시오.

모범답안: PDP는 특정 feature가 평균적으로 prediction에 어떤 영향을 주는지 직관적으로 보여주므로 global explanation에 유용하다. 예를 들어 온도 증가가 자전거 대여량에 평균적으로 어떤 영향을 주는지 확인할 수 있다. 하지만 평균 curve이기 때문에 관측치별 이질성이나 feature interaction을 가릴 수 있고, 서로 다른 집단에서 효과가 반대로 나타나는 경우를 놓칠 수 있다.

문제 55

ICE와 Centered ICE를 비교하시오.

모범답안: ICE는 각 instance마다 관심 feature 변화에 따른 prediction curve를 따로 그려 PDP가 가리는 heterogeneous effect를 확인하게 해준다. 그러나 각 curve의 시작점이 다르면 절대적인 prediction 수준 차이 때문에 feature 변화 효과가 잘 보이지 않을 수 있다. Centered ICE는 기준점에서 curve를 맞춰 상대적 변화량을 보므로, feature가 증가하거나 감소할 때 예측이 어떻게 변하는지 더 명확히 볼 수 있다.

문제 56

Correlation이 낮다고 해서 변수가 중요하지 않다고 단정하면 안 되는 이유를 설명하시오.

모범답안: Correlation coefficient는 주로 선형 관계를 측정한다. 따라서 x가 증가할 때 y가 단조롭게 증가하거나 감소하지 않고, 특정 구간에서 증가했다가 이후 감소하는 비선형 관계가 있으면 correlation은 낮게 나올 수 있다. 하지만 이 경우에도 x와 y 사이에는 중요한 비선형 관계가 존재할 수 있다. 이런 관계는 Mutual Information처럼 확률분포 기반의 관계 측정으로 더 잘 포착될 수 있다.

  11. Bagging은 일반적으로 variance 감소에 효과가 있다.

  13. Bagging에서는 독립적인 모델을 만들기 위해 데이터셋을 여러 bootstrap으로 나눈다.

  14. Bagging은 복원추출이 아니라 항상 비복원추출만 사용한다.

F. Boosting 기본

  1. Boosting은 weak learner를 순차적으로 학습시키는 앙상블 기법이다.
    정답: O

  2. Boosting의 weak learner들은 서로 완전히 독립적이다.
    정답: X

  3. Boosting은 weak learner들을 모아 strong learner를 만든다.
    정답: O

  4. Boosting에서는 이전 모델이 틀린 관측치가 다음 단계에서 더 중요하게 다뤄질 수 있다.
    정답: O

  5. Boosting은 모든 learner를 동시에 병렬 학습하는 것이 핵심이다.
    정답: X

  6. Boosting은 분류하기 어려운 관측치에 더 집중하는 방식으로 설명할 수 있다.
    정답: O

  7. Boosting의 각 단계에서 새 base model은 이전 단계의 오류를 보완하도록 학습된다.
    정답: O

  8. Boosting은 각 단계를 거치며 통합 모델 F(x)가 점차 강해지도록 만든다.
    정답: O

  9. Boosting은 일반적으로 매우 단순한 base learner를 여러 개 결합한다.
    정답: O

  10. Boosting은 weighted sample 또는 residual을 이용해 다음 learner가 집중할 부분을 정한다.
    정답: O

  11. Boosting의 최종 예측은 learner별 가중치 \(\alpha_m\)가 반영된 weighted aggregation으로 표현될 수 있다.
    정답: O

  12. Boosting에서 \(\alpha_m\)은 이전 iteration의 error를 고려해 계산될 수 있다.
    정답: O

  13. Boosting은 overfitting이 절대 발생하지 않는 알고리즘이다.
    정답: X

  14. Boosting은 training error를 계속 낮출 수 있지만 과하면 일반화 성능 문제가 생길 수 있다.
    정답: O

  15. Bagging은 어려운 관측치에 순차적으로 더 집중하고, Boosting은 bootstrap sample을 독립적으로 만든다.
    정답: X
    해설: 설명이 서로 뒤바뀌었다.

  16. Boosting은 bias 감소 경향이 있는 앙상블 방법으로 설명된다.

G. AdaBoost

  1. AdaBoost는 단순한 로직으로 널리 활용된 boosting 방법론이다.
    정답: O

  2. AdaBoost는 처음에 모든 관측치의 weight를 동일하게 둘 수 있다.
    정답: O

  3. AdaBoost 초기 sample weight는 보통 \(\frac{1}{N}\)처럼 uniform하게 설정된다.
    정답: O

  4. AdaBoost는 현재 weight를 바탕으로 sampling 또는 learner 학습을 수행한다.
    정답: O

  5. AdaBoost는 각 weak learner의 error rate를 이용해 learner 신뢰도 \(\alpha\)를 계산한다.
    정답: O

  6. AdaBoost에서 error rate가 낮을수록 일반적으로 \(\alpha\)가 커진다.
    정답: O

  7. AdaBoost에서 \(\alpha\)는 모든 learner에 항상 같은 값으로 고정된다.
    정답: X

  8. AdaBoost에서 정분류된 관측치의 weight는 감소하고 오분류된 관측치의 weight는 증가한다.
    정답: O

  9. AdaBoost는 오분류 샘플을 다음 learner가 더 많이 보게 만든다.
    정답: O

  10. AdaBoost에서 weight update에는 learner의 신뢰도 \(\alpha\)가 반영된다.
    정답: O

  11. AdaBoost는 weight update 후 normalization을 수행해 weight distribution을 다시 정리한다.
    정답: O

  12. AdaBoost의 binary classification 최종 예측은 \(\mathrm{sign}\left(\sum_t \alpha_t h_t(x)\right)\) 형태로 설명할 수 있다.
    정답: O

  13. AdaBoost에서 weak learner의 weighted error가 0.5보다 크면 랜덤 추측보다 못한 learner로 볼 수 있다.
    정답: O

  14. AdaBoost에서 error가 0.5보다 큰 learner는 항상 가장 높은 신뢰도를 받는다.
    정답: X

  15. AdaBoost에서 반복 횟수 T와 실제 채택된 learner 수는 상황에 따라 다를 수 있다.
    정답: O

  16. AdaBoost는 맞춘 샘플의 weight를 높이고 틀린 샘플의 weight를 낮춘다.
    정답: X

  17. AdaBoost는 모델이 틀린 샘플을 데이터셋에서 무조건 삭제한다.
    정답: X

  18. AdaBoost의 sample weight는 다음 단계의 학습 분포에 영향을 준다.
    정답: O

  19. AdaBoost는 binary class classification 의사코드에서 label을 \(-1\), \(+1\)로 표현할 수 있다.
    정답: O

  20. AdaBoost는 learner별 성능 차이를 최종 결합에서 반영하지 않는다.
    정답: X

H. Gradient Boosting Machine

  1. Gradient Boosting은 Gradient Descent와 Boosting을 결합한 관점으로 설명된다.
    정답: O

  2. GBM은 현재 모델이 설명하지 못한 residual을 줄이는 방향으로 새 learner를 추가한다.
    정답: O

  3. Residual은 y - F(x)처럼 실제값과 현재 모델 예측의 차이로 볼 수 있다.
    정답: O

  4. GBM은 숫자 parameter 하나만 업데이트하고 function은 추가하지 않는다.
    정답: X

  5. Functional gradient descent 관점에서 GBM은 기존 ensemble function에 새 function을 더한다.
    정답: O

  6. GBM은 모델 형태를 하나로 고정한 뒤 coefficient만 추정하는 선형회귀와 같은 방식이다.
    정답: X

  7. GBM에서는 \(F_{k+1}(x) = F_k(x) + alpha * f_{k+1}(x)\) 형태로 새 learner를 반영할 수 있다.
    정답: O

  8. GBM의 learning rate \(\alpha\)는 gradient 업데이트 양 또는 새 learner 반영 정도를 조절한다.
    정답: O

  9. Learning rate가 너무 크면 최적점을 지나치거나 학습이 불안정해질 수 있다.
    정답: O

  10. Learning rate가 너무 작으면 수렴이 느려질 수 있다.
    정답: O

  11. GBM은 직전 단계의 오차를 학습하는 boosting 방식으로 설명된다.
    정답: O

  12. AdaBoost는 오분류 관측치 가중치를 올리는 방식이고, Gradient Boosting은 residual/loss를 줄이는 방식으로 설명할 수 있다.
    정답: O

  13. GBM의 weak learner 개수는 생성하는 tree 수와 관련된 hyperparameter이다.
    정답: O

  14. GBM에서 tree 수가 불필요하게 너무 크면 학습 속도가 저하될 수 있다.
    정답: O

  15. GBM의 max depth는 tree weak learner의 복잡도를 조절한다.
    정답: O

  16. GBM에서 max features가 커지면 tree 간 상관관계가 증가해 random subspace 효과가 줄어들 수 있다.
    정답: O

  17. GBM의 min samples leaf는 최종 leaf node에 포함되는 관측치 수를 조정한다.
    정답: O

  18. GBM의 min samples split은 tree 분기 시 최소한 포함해야 하는 관측치 수와 관련된다.
    정답: O

  19. GBM은 overfitting 문제가 전혀 없기 때문에 regularization이나 depth 조절이 필요 없다.
    정답: X

  20. GBM은 대다수 boosting 기법의 근간 또는 문제의식을 제공하는 모델로 볼 수 있다.
    정답: O

I. XGBoost

  1. XGBoost는 Extreme Gradient Boosting의 약자이다.
    정답: O

  2. XGBoost는 GBM의 과적합 문제와 학습 속도 문제를 보완하려는 알고리즘으로 소개된다.
    정답: O

  3. XGBoost는 GBM + regularization and parallelism으로 요약할 수 있다.
    정답: O

  4. XGBoost는 regularization을 전혀 사용하지 않는 순수 GBM이다.
    정답: X

  5. XGBoost의 목적식은 loss term과 regularization term을 함께 포함할 수 있다.
    정답: O

  6. XGBoost의 regularization은 weak learner 복잡도와 leaf weight를 제어하는 데 사용된다.
    정답: O

  7. XGBoost는 개별 tree가 pruning될 기회를 늘려 일반화 성능을 높이려 한다.
    정답: O

  8. XGBoost에서 split gain이 \(\gamma\)보다 작으면 pruning 대상이 될 수 있다.
    정답: O

  9. XGBoost에서 \(\gamma\)는 split을 무조건 더 많이 만들도록 하는 값이다.
    정답: X

  10. XGBoost의 lambda는 leaf output 계산에서 regularization 역할을 할 수 있다.
    정답: O

  11. XGBoost에서 residual 부호가 같은 방향으로 많이 몰리면 similarity가 커질 수 있다.
    정답: O

  12. Residual 부호가 서로 상쇄되는 zero-sum 상황은 similarity를 줄이는 방향으로 설명된다.
    정답: O

  13. XGBoost는 approximate split finding, sparsity-aware split finding 같은 split 관련 전략을 사용할 수 있다.
    정답: O

  14. XGBoost는 column block이나 cache-aware block structure 등 계산 효율 개선을 고려한다.
    정답: O

  15. XGBoost의 parallelism은 단순히 이름에 X가 붙었다는 뜻이다.
    정답: X

  16. XGBoost는 tabular dataset의 분류와 예측 문제에서 강한 성능으로 많이 활용된다.
    정답: O

  17. XGBoost는 top numerical feature 중심으로 높은 정확도가 필요할 때 추천될 수 있다.
    정답: O

  18. XGBoost는 categorical feature를 CatBoost처럼 자동으로 전용 처리하는 데 초점이 있다.
    정답: X

  19. XGBoost의 주요 hyperparameter에는 weak learner 수, learning rate, max depth 등이 포함된다.
    정답: O

  20. XGBoost의 max features가 크면 tree 간 상관 증가로 random subspace 효과가 줄어들 수 있다.
    정답: O

  21. XGBoost는 leaf weight의 L2 regularization 같은 방식으로 과적합을 억제할 수 있다.
    정답: O

J. LightGBM

  1. LightGBM은 정확도 못지않게 학습 속도 개선에 초점을 둔 boosting 알고리즘이다.
    정답: O

  2. LightGBM은 기존 boosting과 XGBoost의 단점을 함께 보완하려는 배경에서 설명된다.
    정답: O

  3. LightGBM의 대표 특징에는 GOSS와 EFB가 있다.
    정답: O

  4. GOSS는 Gradient-based One-sided Sampling의 약자이다.
    정답: O

  5. GOSS는 큰 gradient를 갖는 관측치는 유지하고 작은 gradient 관측치는 일부 제거/샘플링한다.
    정답: O

  6. GOSS는 gradient가 큰 샘플을 중요하지 않다고 보고 먼저 제거한다.
    정답: X

  7. EFB는 Exclusive Feature Bundling의 약자이다.
    정답: O

  8. EFB는 sparse dataset에서 상호배타적인 feature를 bundling해 feature 수를 줄인다.
    정답: O

  9. EFB는 PCA처럼 모든 feature를 연속적인 주성분으로 바꾸는 방법이다.
    정답: X

  10. LightGBM은 leaf-wise로 tree를 성장시킬 수 있다.
    정답: O

  11. LightGBM의 leaf-wise growth는 불균형 tree를 만들 수 있다.
    정답: O

  12. LightGBM은 항상 depth-wise로만 tree를 균등하게 성장시킨다.
    정답: X

  13. LightGBM은 학습 속도가 중요한 제약일 때 추천될 수 있다.
    정답: O

  14. LightGBM은 categorical feature를 입력받기 위한 특수 알고리즘을 제공할 수 있다.
    정답: O

  15. LightGBM은 GOSS를 통해 optimal split point 탐색을 돕는다고 설명된다.
    정답: O

  16. LightGBM은 boosting 계열이 아니며 linear regression 전용 알고리즘이다.
    정답: X

K. CatBoost

  1. CatBoost는 Categorical Boost의 약자이다.
    정답: O

  2. CatBoost는 범주형 변수를 자체 처리할 수 있는 장점이 있다.

  3. CatBoost는 target leakage와 prediction shift 문제에 초점을 둔다.

  5. Target encoding은 잘못 사용하면 test 시 모르는 y 정보를 feature 생성에 쓰는 target leakage를 일으킬 수 있다.

  6. One-hot encoding은 cardinality가 높은 categorical feature에서 dataset을 지나치게 sparse하게 만들 수 있다.
    정답: O

  7. CatBoost의 Ordered Target Statistics는 현재 모르는 미래 target 값을 사용하지 않도록 인코딩한다.

    정답: X

  9. CatBoost는 종속변수와 유사한 분류 정보를 갖는 중복 categorical feature를 제외하는 처리도 언급된다.
    정답: O

  10. Ordered Boosting은 학습 데이터와 테스트 데이터 간 분포 차이로 생기는 prediction shift를 줄이려는 방식이다.
    정답: O

  11. CatBoost는 categorical feature가 많은 데이터에서 추천될 수 있다.
    정답: O

  12. CatBoost는 categorical feature를 전혀 처리하지 못하므로 사용자가 모두 수동 처리해야 한다.
    정답: X

  13. CatBoost는 accuracy와 training speed가 모두 중요하고 categorical feature 비율이 큰 경우 추천될 수 있다.
    정답: O

  14. CatBoost는 tree를 level-wise로 성장시켜 balanced tree를 만들 수 있다고 비교표에서 설명된다.
    정답: O

  15. CatBoost의 MVS는 split accuracy를 높이는 데 도움을 주는 방식으로 비교표에 등장한다.
    정답: O

L. XGBoost vs LightGBM vs CatBoost 비교

  1. XGBoost, LightGBM, CatBoost는 모두 boosting 계열로 tabular dataset에서 자주 활용된다.
    정답: O

  2. XGBoost는 regularization/parallelism, LightGBM은 속도, CatBoost는 categorical feature 처리에 강점이 있다.
    정답: O

  3. 세 알고리즘은 이름만 다르고 학습 전략과 장점이 완전히 동일하다.
    정답: X

  4. XGBoost는 categorical feature를 사용자가 수동 처리해야 하는 경우가 많다고 비교된다.
    정답: O

  5. LightGBM은 training speed가 가장 큰 제약일 때 고려할 수 있다.
    정답: O

  6. CatBoost는 categorical feature가 많은 데이터에서 고려할 수 있다.
    정답: O

  7. XGBoost는 tree를 depth 방향으로 성장시키고, LightGBM은 leaves 방향으로 성장시키는 것으로 비교된다.
    정답: O

  8. LightGBM의 leaf-wise 방식은 언제나 완전히 balanced tree만 만든다.
    정답: X

  9. 세 알고리즘은 모두 최근 데이터 컨테스트에서 자주 활용될 정도로 실용성이 높다.
    정답: O

  10. XGBoost, LightGBM, CatBoost는 실제 데이터에서는 비교 실험을 통해 선택하는 것이 일반적이다.
    정답: O

M. Stacking

  3. Stacking은 단순 voting이나 평균만 수행하는 방식과 동일하다.

  4. Stacking은 다양성을 극대화하기 위해 서로 다른 알고리즘의 weak learner를 사용할 수 있다.

A. Ensemble과 Bias-Variance

  1. Ensemble은 여러 base learner의 예측을 결합해 단일 모델보다 오류를 줄이려는 접근이다.
    정답: O

  2. Ensemble 효과를 내려면 모든 base learner가 완전히 같은 예측을 하는 것이 가장 좋다.
    정답: X. diversity가 중요하다.

  3. Bagging은 bootstrap sample로 learner를 독립적으로 학습시키고 aggregate해 variance를 줄이는 데 초점이 있다.
    정답: O

  4. Boosting은 learner들을 순차적으로 학습시키며 이전 learner의 오류나 residual을 보완한다.
    정답: O

  5. Stacking은 base learner의 예측을 meta learner가 다시 학습하는 구조다.
    정답: O

  6. Bias는 모델 예측 평균이 실제 정답에서 체계적으로 벗어나는 정도다.
    정답: O

  7. Variance는 training data가 달라질 때 모델 예측이 흔들리는 정도다.
    정답: O

  8. Optimal complexity는 항상 bias와 variance가 정확히 교차하는 지점이다.
    정답: X. 전체 expected error가 최소가 되는 지점이다.

  9. No Free Lunch theorem은 모든 문제에서 항상 최고인 단일 알고리즘은 없다는 뜻이다.
    정답: O

  10. 딥러닝이나 앙상블은 No Free Lunch theorem을 완전히 무효화했다.
    정답: X

B. Random Forest, AdaBoost, GBM

  1. Random Forest는 supervised learning에 사용할 수 있다.
    정답: O

  2. Random Forest의 핵심은 bootstrap sampling과 random subspace/feature subsampling이다.
    정답: O

  3. Random Forest는 boosting처럼 loss를 반복 최적화하면서 weak learner를 순차적으로 추가한다.
    정답: X

  4. Random Forest에서 tree 생성은 독립 시행에 가깝지만 같은 원 데이터 분포를 따르므로 완전히 무관한 모델이라는 뜻은 아니다.
    정답: O

  5. Random Forest 정확도 향상 선지에서 비용/시간 제약이 없다면 학습 데이터 추가 확보는 안정적인 답이 될 수 있다.
    정답: O

  6. AdaBoost는 오분류 sample weight를 올리고 정분류 sample weight를 낮춘다.
    정답: O

  7. AdaBoost에서 weighted error가 0.5 이상인 binary weak learner는 랜덤보다 못하므로 버리거나 중단할 수 있다.
    정답: O

  8. AdaBoost의 최종 예측은 learner 신뢰도 \(\alpha_t\)가 반영된 weighted sign aggregation으로 볼 수 있다.
    정답: O

  9. GBM은 residual/loss를 줄이는 방향으로 weak learner를 순차적으로 추가한다.
    정답: O

  10. Functional Gradient Descent 관점에서 GBM은 숫자 parameter만 업데이트하고 function은 추가하지 않는다.
    정답: X. loss를 줄이는 함수/weak learner를 추가한다.

  11. Boosting의 weak learner가 지나치게 복잡하면 과적합 위험이 커질 수 있다.
    정답: O

  12. Learning rate는 새 learner 또는 gradient 방향을 얼마나 반영할지 조절한다.
    정답: O

  9. XGBoost, LightGBM, CatBoost는 이름만 다르고 학습 전략과 강점이 완전히 같다.

  12. DART는 boosting step에서 일부 tree를 dropout해 과적합을 줄이려는 아이디어를 사용한다.
    정답: O

  13. NGBoost는 예측값 하나만이 아니라 분포와 불확실성까지 모델링하려는 boosting 계열 확장이다.
    정답: O

D. 성능평가지표와 검증

  5. MSE/RMSE는 큰 오차에 더 민감하다.
    정답: O

  6. MAPE는 실제값이 0 또는 0에 가까울 때 안정적이다.

  7. MASE가 1보다 작으면 naive forecast보다 좋은 것으로 해석할 수 있다.

  8. R square는 항상 0과 1 사이에만 존재한다.
    정답: X. 나쁜 모델에서는 음수도 가능하다.

  9. Confusion matrix로 지표를 계산하기 전 actual/predicted 축 방향을 먼저 확인해야 한다.
    정답: O

  10. Recall은 \(\frac{TP}{TP+FN}\)이다.
    정답: O

  11. Precision은 \(\frac{TP}{TP+FP}\)이다.
    정답: O

  12. Specificity는 \(\frac{TN}{TN+FP}\)이다.
    정답: O

  13. F1 score는 precision과 recall의 조화평균이다.
    정답: O

  14. ROC curve의 x축은 FPR, 즉 1-specificity, y축은 TPR/sensitivity다.
    정답: O

  15. K-fold cross validation은 test set을 train set에 섞어 최종 평가를 여러 번 하는 방법이다.
    정답: X. validation fold를 바꿔가며 평가한다.

  11. Global explanation은 모델 전체 경향, local explanation은 특정 관측치의 예측 이유를 설명한다.

  3. ICE는 instance별 prediction curve를 보여준다.
    정답: O

  4. Centered ICE는 시작점을 맞춰 절대 수준 차이보다 상대 변화량을 보기 쉽게 한다.
    정답: O

  5. Morris sensitivity analysis는 one-at-a-time 방식으로 feature를 변화시킨다.
    정답: O

  6. Morris의 \(\mu\)는 평균적 영향력, \(\sigma\)는 변동성이나 interaction/nonlinearity 가능성을 나타낸다.
    정답: O

  7. Permutation importance는 feature 값을 섞어 정보를 깨뜨린 뒤 성능 하락량으로 중요도를 본다.
    정답: O

  8. 상관된 feature가 있으면 permutation importance 해석이 왜곡될 수 있다.
    정답: O

  9. Correlation이 낮으면 비선형 관계도 절대 존재하지 않는다.

  10. Mutual information은 확률분포 기반으로 비선형 dependence도 포착할 수 있다.

  11. KL divergence는 두 분포 차이나 정보 손실과 관련된다.

  5. LIME은 global fidelity를 보장하므로 전체 데이터 분포에서 원 모델을 항상 정확히 대체한다.

  9. SHAP은 Shapley value와 cooperative game theory 기반의 additive feature attribution 방법이다.

  10. SHAP에서 각 feature는 game의 player처럼 볼 수 있다.
    정답: O

  11. Local accuracy는 feature contribution 합이 원 모델 예측과 일치한다는 성질이다.

  12. Missingness는 존재하지 않는 feature의 attribution이 0이라는 성질이다.

  13. Consistency는 feature의 marginal contribution이 커졌는데 attribution이 줄어드는 것을 허용한다.

  14. SHAP summary plot은 feature importance, feature value high/low, 영향 방향과 분포를 함께 볼 수 있다.

  15. SHAP dependence plot은 feature value와 SHAP value 관계를 볼 수 있다.

  16. SHAP clustering은 instance의 Shapley value 유사성을 이용해 비슷한 설명 패턴을 묶을 수 있다.

범위: Ensemble II, 성능평가지표, XAI I-IV, 최적화

A. 서술형

문제 1

Bagging, Boosting, Stacking을 learner 관계와 결합 방식 중심으로 비교하시오.

모범답안: Bagging은 bootstrap sample로 여러 learner를 대체로 독립적으로 학습하고 voting/average로 결합해 variance를 줄인다. Boosting은 이전 learner의 오류나 residual을 다음 learner가 보완하도록 순차적으로 학습하며 bias를 줄이는 경향이 있다. Stacking은 여러 base learner의 예측을 meta learner가 다시 학습해 결합하는 구조이며, heterogeneous learner를 쓰면 서로 다른 알고리즘의 장점을 취할 수 있다.

채점 키워드: bootstrap, 독립, sequential, residual/error, meta learner

문제 2

Random Forest가 단순 decision tree 집합과 구분되는 핵심을 쓰시오.

모범답안: Random Forest는 decision tree를 여러 개 모은 것만이 아니라, bootstrap sampling으로 서로 다른 training subset을 만들고 random subspace/feature subsampling으로 split 후보 feature를 제한해 tree 다양성을 높인다. 이런 다양성을 aggregate하면서 variance를 줄이고 일반화 성능을 높인다.

채점 키워드: bootstrap, bagging, random subspace, feature subsampling, diversity

문제 3

AdaBoost의 학습 절차를 간단히 설명하시오.

모범답안: 처음에는 sample weight를 균일하게 둔다. 각 반복에서 현재 weight 분포로 weak learner를 학습하고 weighted error를 계산한다. error가 낮은 learner에는 큰 \(\alpha\)를 주고, 오분류 sample weight는 증가시키며 정분류 sample weight는 감소시킨 뒤 normalization한다. 최종 예측은 \(\mathrm{sign}\left(\sum_t \alpha_t h_t(x)\right)\) 형태의 weighted aggregation으로 한다.

채점 키워드: sample weight, weighted error, alpha, 오분류 weight 증가, sign aggregation

문제 4

GBM을 residual/loss와 Functional Gradient Descent 관점에서 설명하시오.

모범답안: GBM은 현재 ensemble이 설명하지 못한 residual 또는 loss를 줄이는 방향으로 weak learner를 순차적으로 추가한다. 일반 gradient descent가 숫자 parameter를 업데이트한다면, Functional Gradient Descent 관점에서는 loss를 줄이는 함수, 즉 새로운 weak learner를 더해 ensemble function을 개선한다.

채점 키워드: residual, loss, weak learner, 순차, function 추가

문제 5

XGBoost가 GBM에서 개선한 점을 regularization과 계산 효율 측면에서 설명하시오.

모범답안: XGBoost는 GBM의 loss를 줄이는 구조에 regularization term을 추가해 tree 복잡도와 leaf weight가 과도하게 커지는 것을 억제한다. gamma를 이용한 split/pruning 제어와 L1/L2 penalty가 대표적이다. 또한 sorted feature, gradient statistics 재사용, cache-aware block structure 등으로 계산 효율과 병렬성을 개선한다.

채점 키워드: objective, regularization, gamma, L1/L2, parallel/cache/block

문제 6

LightGBM과 CatBoost의 핵심 강점을 비교하시오.

모범답안: LightGBM은 leaf-wise growth, GOSS, EFB를 통해 학습 속도와 효율성을 높이는 데 강점이 있다. CatBoost는 categorical feature 처리에 강하고, target encoding에서 발생할 수 있는 target leakage와 prediction shift를 ordered target statistics와 ordered boosting으로 완화한다.

채점 키워드: LightGBM, leaf-wise, GOSS, EFB, CatBoost, ordered target statistics

문제 7

회귀 지표 MAE, MSE, RMSE, MAPE, MASE, R square의 핵심 차이를 쓰시오.

모범답안: MAE는 절대오차 평균이라 원래 단위로 해석하기 쉽다. MSE는 오차 제곱 평균이라 큰 오차에 민감하고 단위가 제곱된다. RMSE는 MSE의 제곱근이라 원래 단위로 해석하기 쉽지만 큰 오차에 민감하다. MAPE는 percentage error라 직관적이지만 실제값이 0에 가까우면 불안정하다. MASE는 naive forecast 대비 성능을 본다. R square는 전체 변동 중 모델이 설명한 비율이며 나쁜 모델에서는 음수도 가능하다.

채점 키워드: 절대오차, 제곱오차, percentage, naive, R square 음수 가능

문제 8

Precision, Recall, Specificity, F1 score를 confusion matrix로 설명하시오.

모범답안: Recall은 실제 positive 중 모델이 positive로 잡은 비율로 \(\frac{TP}{TP+FN}\)이다. Precision은 positive라고 예측한 것 중 실제 positive인 비율로 \(\frac{TP}{TP+FP}\)이다. Specificity는 실제 negative 중 negative로 맞춘 비율로 \(\frac{TN}{TN+FP}\)이다. F1 score는 precision과 recall의 조화평균으로, 두 지표를 함께 고려한다. 계산 전에는 confusion matrix의 actual/predicted 행열 방향을 확인해야 한다.

채점 키워드: TP/FN/FP/TN, recall, precision, specificity, F1, 축 확인

문제 9

XAI가 필요한 이유를 accuracy, overconfidence, fairness, accountability 관점에서 설명하시오.

모범답안: 모델 accuracy가 높아도 실제 현장에서는 왜 그런 예측을 했는지 설명해야 한다. 최신 모델은 틀린 예측에도 높은 confidence를 줄 수 있어 overconfidence 문제가 생긴다. 의료, 금융, 법률 같은 분야에서는 편향과 fairness, 책임 소재, 의사결정 근거가 중요하다. XAI는 모델이 어떤 feature에 근거해 판단했는지 보여주어 신뢰, 디버깅, 데이터 개선에 도움을 준다.

채점 키워드: accuracy 한계, overconfidence, fairness, accountability, debugging

문제 10

Intrinsic interpretability와 post-hoc interpretability를 \(f(x)\), g(x) 관점에서 설명하시오.

모범답안: Intrinsic interpretability는 linear regression이나 decision tree처럼 예측 모델 \(f(x)\) 자체가 사람이 이해하기 쉬운 구조인 경우다. Post-hoc interpretability는 먼저 black-box 예측 모델 \(f(x)\)를 학습한 뒤, 별도의 설명 모델 또는 방법 g(x)로 \(f(x)\)의 예측 근거를 해석하는 방식이다. 이때 g(x)는 예측 성능보다 사람이 이해할 수 있는 설명을 만드는 것이 목적이다.

채점 키워드: intrinsic, post-hoc, \(f(x)\), g(x), black-box

문제 11

PDP, ICE, Centered ICE를 비교하시오.

모범답안: PDP는 관심 feature를 변화시키고 나머지 feature를 marginalization하여 평균 예측 변화를 보여주는 global explanation이다. ICE는 각 instance별 prediction curve를 보여주어 PDP가 가릴 수 있는 heterogeneous relation을 확인하게 해준다. Centered ICE는 curve 시작점을 기준점에 맞춰 절대 예측 수준 차이보다 feature 변화에 따른 상대 변화량을 보기 쉽게 한다.

채점 키워드: marginal effect, global, instance curve, heterogeneity, centered

문제 12

Morris sensitivity analysis와 Permutation importance를 비교하시오.

모범답안: Morris는 다른 feature를 고정하고 하나의 feature만 변화시키는 one-at-a-time sensitivity analysis다. \(\mu\)는 평균적 영향력, \(\sigma\)는 영향의 변동성이나 interaction/nonlinearity 가능성을 나타낸다. Permutation importance는 특정 feature 값을 섞어 정보력을 깨뜨린 뒤 성능 하락량으로 중요도를 판단한다. 둘 다 model-agnostic하게 쓸 수 있지만, permutation은 상관 feature가 있을 때 해석이 왜곡될 수 있다.

채점 키워드: OAT, \(\mu\), \(\sigma\), feature shuffle, 성능 하락, 상관 feature

문제 13

EBM을 GAM/GA2M과 boosting 관점에서 설명하시오.

모범답안: GAM은 feature별 변환 함수의 additive combination으로 예측하는 해석 가능한 모델이지만 feature interaction을 기본적으로 놓칠 수 있다. GA2M은 pairwise interaction을 일부 추가한다. EBM은 이런 GAM/GA2M 구조에 gradient boosting을 적용해 feature별 함수와 일부 interaction을 학습하는 glass-box 모델이다. feature 하나씩 tree를 반복 학습하고 feature-wise contribution graph로 해석할 수 있다.

채점 키워드: GAM, additive, interaction, boosting, glass-box, contribution graph

문제 14

LIME의 기본 절차와 한계를 설명하시오.

모범답안: LIME은 설명하고 싶은 data point 주변의 입력을 perturbation하고, black-box model f의 예측을 얻은 뒤, 그 주변에서 f를 근사하는 단순하고 sparse한 surrogate model g를 학습한다. 이미지에서는 superpixel을 켜고 끄는 binary encoding을 쓸 수 있고, text에도 적용 가능하다. 한계는 neighborhood 정의, discretization, 충분한 perturbation coverage, random sampling 안정성, 시간 비용 문제다.

채점 키워드: local perturbation, surrogate, sparse linear/LASSO, superpixel, local fidelity, 한계

문제 15

SHAP을 Shapley value와 additive feature attribution 관점에서 설명하시오.

모범답안: SHAP은 각 feature를 cooperative game의 player로 보고, 특정 prediction에 대한 feature별 공정한 기여도인 Shapley value를 계산한다. 설명은 baseline prediction에서 각 feature contribution을 더해 실제 예측을 만드는 additive feature attribution 형태다. Local accuracy, missingness, consistency 같은 성질을 가지며, local explanation과 global summary를 함께 제공할 수 있다.

채점 키워드: cooperative game, feature as player, baseline, additive, local accuracy/missingness/consistency

문제 16

KernelSHAP, TreeSHAP, DeepSHAP/Integrated Gradient SHAP을 구분하시오.

모범답안: KernelSHAP은 model-agnostic하게 SHAP을 근사하지만 data point 수가 많으면 느리고 feature dependence를 무시할 수 있다. TreeSHAP은 tree-based model에서 빠르게 SHAP value를 계산하는 구현이지만 직관적이지 않은 attribution이 나올 수 있다. DeepSHAP과 Integrated Gradient SHAP은 deep learning이나 image model 설명 예시로 등장한다.

채점 키워드: KernelSHAP, model-agnostic/느림, TreeSHAP/tree/빠름, DeepSHAP, Integrated Gradient

C. 계산형

문제 1. Confusion matrix 지표 계산

어떤 이진 분류 모델에서 \(TP=40\), \(FP=10\), \(FN=20\), \(TN=30\)이다. Accuracy, Recall, Precision, Specificity, F1 score를 구하시오.

풀이:

\[ \begin{aligned} Accuracy &= (TP+TN)/(TP+FP+FN+TN) = (40+30)/100 = 0.70\\ Recall &= TP/(TP+FN) = 40/(40+20) = 0.667\\ Precision &= TP/(TP+FP) = 40/(40+10) = 0.80\\ Specificity &= TN/(TN+FP) = 30/(30+10) = 0.75\\ F_1 &= 2PR/(P+R) = 2*0.8*0.667/(0.8+0.667) = 0.727 \end{aligned} \]

정답: Accuracy 0.70, Recall 0.667, Precision 0.80, Specificity 0.75, F1 약 0.727

문제 2. 회귀 지표 계산

실제값 [10, 20, 30], 예측값 [12, 18, 33]일 때 MAE와 MSE, RMSE를 구하시오.

풀이:

\[ \begin{aligned} errors &= [2, -2, 3]\\ absolute errors &= [2, 2, 3]\\ squared errors &= [4, 4, 9]\\ \\ MAE &= (2+2+3)/3 = 2.333\\ MSE &= (4+4+9)/3 = 5.667\\ RMSE &= \sqrt(5.667) = 2.381 \end{aligned} \]

정답: MAE 약 2.33, MSE 약 5.67, RMSE 약 2.38

문제 3. FPR과 ROC 좌표

\(TP=70\), \(FN=30\), \(FP=20\), \(TN=80\)일 때 ROC curve에 찍히는 좌표 \((FPR, TPR)\)를 구하시오.

풀이:

\[ \begin{aligned} TPR &= Recall = TP/(TP+FN) = 70/(70+30) = 0.70\\ FPR &= FP/(FP+TN) = 20/(20+80) = 0.20 \end{aligned} \]

정답: (0.20, 0.70)

문제 1

Gradient descent의 기본 업데이트 식과 각 기호의 의미를 쓰시오.

모범답안:

\[ w \leftarrow w-\eta\nabla J(w) \]

\(w\)는 현재 파라미터, \(\eta\)는 learning rate, \(\nabla J(w)\)는 목적함수 \(J\)의 gradient이다. 목적함수를 줄이기 위해 gradient의 반대 방향으로 파라미터를 이동한다.

문제 2

ERM이 과적합에 취약한 이유를 설명하시오.

모범답안:
ERM은 unknown data-generating distribution 대신 현재 학습데이터에서의 empirical risk를 줄인다. 따라서 모델 복잡도가 크거나 데이터가 작으면 학습데이터의 noise까지 맞추면서 training error는 낮아지지만 validation/test error는 증가할 수 있다.

문제 3

Grid search와 random search를 비교하시오.

모범답안:
Grid search는 정해진 격자 조합을 모두 평가하는 exhaustive search라 단순하지만 hyperparameter 수가 늘면 계산량이 폭증한다. Random search는 임의 조합을 평가하므로 전역 최적을 보장하지는 않지만 제한된 자원에서 더 효율적인 탐색이 가능할 수 있다.

문제 4

Heuristic과 metaheuristic의 차이를 설명하시오.

모범답안:
Heuristic은 시간 또는 자원 제약에서 충분히 좋은 근사해를 빠르게 찾기 위한 문제 해결 규칙이다. Metaheuristic은 특정 문제에 직접 종속되지 않는 상위 탐색 전략으로, simulated annealing이나 genetic algorithm처럼 global optimum 보장보다 제한 시간 내 suboptimal solution 탐색을 목표로 한다.

AdaBoost 의사코드를 작성하시오.

모범답안:

Input: training data, weak learner, number of learners T
Initialize sample weights D_i = 1/n
For t = 1 to T:
    Train weak learner h_t using D
    Compute weighted error epsilon_t
    Compute learner weight alpha_t
    Increase weights of misclassified samples
    Decrease weights of correctly classified samples
    Normalize D
Output weighted vote of weak learners

핵심 식:

\[ \alpha_t=\frac{1}{2}\ln\frac{1-\epsilon_t}{\epsilon_t} \]
